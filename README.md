# Elm-Node Boiler Plate

### Make your lols app in half the time:
```
thinkOfAJoke
  |> implementJoke
  |> sitBackAndEnjoy
```

### Stack
- Node server
- Postgres DB
- Elm frontend
- Style Elements for never having to write css again!

### Things you might need:
config.env
```
DATABASE_URL=
```

### Get it setup:
```
npm i
cd client && elm-package install
```

### Get it running:
```
npm run dev
(in another terminal)
npm run webpack
```
